1. 문자와 문자열의 차이

  • 'A' : 문자, 한글자만 사용, 작은따옴표
  • "ABC" : 문자열, 여러 글자 사용, 큰따옴표

2. 문자열 사용

1) 포인터 사용

char* p = "abcde";

  • 출력할때 %s 사용
  • "abcde"는 문자열리터럴
  • p는 문자열의 주소를, p*는 문자열의 아스키코드를 가리킴
  • 인덱스로 접근 : p[0]~p[5]로 문자열 하나씩 읽기 가능 %c 사용
  • 읽기전용 - 덮어쓰기 불가능 / 동적할당시 쓰기 가능
  • 문자열의 마지막에 끝을 나타내는 널 문자 붙음 NULL == \0
*코드 1

2) 배열 사용

char str[6] = "abcde";

  • 최소 크기 : 문자열 크기 + null까지 포함한 크기
  • 배열 인덱스에 문자 하나씩 넣어줌
  • 출력은 포인트와 같은 방법 : printf("%s",str);
  • 인덱스로 하나씩 접근 : str[0]~str[5] printf("%c", str[0])
  • char str[] = "abc"; 처럼 크기 생략 가능
  • 배열 선언 즉시 문자열로 초기화 해줘야한다.
    *코드 2
  • 이미 선언된 배열에 문자열을 할당하고 싶으면 1) 인덱스 요소에 하나씩 넣어주거나 2) strcpy 함수를 사용
    *코드 3

3. 문자열 입력받기

1) 포인터 사용

char *p = malloc(sizeof(char) * 10));

scanf("%s", &p);

  • 포인터는 꼭 공간 만들어주기 > 나중에 free() 해주기!

4. 문자열 함수

1) strlen : 문자열 길이

2) strcmp(a,b) : 문자열 비교 / 같으면 0

- a>b면 1, 반대면 -1, 같으면 0

- ==와의 비교 설명하기 (==은 주소값이 달라서 결국 같다고 비교 못함)

3) strcpy(a, b) : a를 b에 복사

- 배열>배열은 괜찮은데 배열 > 포인터는 X(동적할당 필요)

4) strcat(a, b) : a에 b붙이기

5) sprintf(배열, 서식, 값) : 배열에 저장

- 배열 또는 동적할당해준 포인터 사용 가능

- 서식에 %d %c %f 등 다양한 자료형으로 문자열을 만듦

6) strchr(배열, 'a') : 배열에서 a 검색 - 포인터 반환

- 포인터에서 다음에 오는 문자를 찾아준다

7) strrchr(문자열, 검색할 문자) : 오른쪽부터 문자 찾기

8) strstr(문자열, "den") : 문자열 찾기

9) strtok(대상문자열, "기준 문자") : 문자열 자르기

- 보통 while문과 많이 사용

- strtok(NULL, "기준문자") : 기준문자를 널 문자로 채워줌

- Apple(\n)banana(\n)melon
